## 1. TIKTOK — ALGORITHME 2026

### Ce que TikTok mesure (par ordre d'importance)

| Signal | Poids | Objectif Lucas |
|--------|-------|----------------|
| Watch time / rétention | ⭐⭐⭐⭐⭐ | > 60% à 5s, > 20% complétion |
| Partages | ⭐⭐⭐⭐⭐ | ×5 vs like |
| Sauvegardes | ⭐⭐⭐⭐ | ×4 vs like |
| Commentaires | ⭐⭐⭐ | ×2 vs like |
| Likes | ⭐ | Signal secondaire — ne pas optimiser |

**Règle absolue** : 0 CTA actif = bloqué à 300 vues même avec un bon hook.

### Distribution TikTok

```
Étape 1 — Batch test (~200-500 viewers non-abonnés)
         ↓ complétion < 20% → vidéo enterrée
         ↓ complétion > 20% → étape 2

Étape 2 — Distribution élargie (commentaires + partages = carburant)
Étape 3 — Si partages forts → viral
```

### Les 3 paliers (validé terrain @chronovision.fr)
| Palier | Signal N°1 | Signal N°2 | Seuil critique |
|--------|------------|------------|----------------|
| 0 → 300 | Complétion > 20% | Rétention 5s > 50% | < 20% = vidéo enterrée |
| 300 → 1 000 | Partages + commentaires actifs | 3 CTAs actifs | 0 CTA = bloqué ici |
| 1 000 → viral | CTR onglet "Pour Toi" | Partages massifs | Dépend du twist |

## 2. TIKTOK — STRUCTURE DU SCRIPT (niche agents IA Claude)

### Format cible : 30-60 secondes

```
HOOK (0-3s) — arrêter le scroll
WTF MOMENT (3-8s) — révélation immédiate
DÉVELOPPEMENT (8-40s) — valeur dense ou démo express
CTA EMPILÉS (40-60s) — 3 CTAs distincts
```

### Types de contenu TikTok (niche agents IA)

**Type A — Démo flash (le plus fort)**
Montrer l'agent en action en 30-60s.
Hook : résultat impressionnant d'abord → expliquer comment ensuite.
```
"Regarde ce que cet agent Claude vient de produire en 8 secondes."
[montrer output]
"Voilà comment il fonctionne."
[30s de démo ultra compressée]
CTA : "Commente AGENT si tu veux le même."
```

**Type B — Info IA (actualité + solution)**
News Claude/IA + application concrète pour la cible.
```
"Claude vient de sortir [feature]. Voilà ce que ça change pour toi si tu crées du contenu."
[30s d'explication + exemple concret]
CTA : "Enregistre ça — tu vas en avoir besoin."
```

**Type C — Clip YouTube (déclinaison)**
Extrait du meilleur moment de la vidéo YouTube.
```
[Moment le plus utile ou impressionnant de la vidéo YT]
"Tuto complet en lien dans ma bio."
```

**Type D — Problème → Solution**
Nommer LE problème de la cible → montrer que l'agent le résout.
```
"Tu passes combien d'heures à écrire tes scripts TikTok chaque semaine ?"
[pause courte]
"J'ai créé un agent qui fait ça en 30 secondes."
[démo]
CTA : "Commente AGENT pour l'avoir."
```

## 3. TIKTOK — RÈGLES DES HOOKS (niche agents IA)

### Formules validées pour la niche

**Formule 1 — Résultat chiffré**
"[L'agent] a produit [X outputs] en [durée]."
Ex : "Cet agent a écrit 5 scripts TikTok en 45 secondes."

**Formule 2 — Gain de temps**
"[Tâche] te prend [durée]. L'IA le fait en [durée]."
Ex : "Écrire un script TikTok te prend 1 heure. Claude le fait en 30 secondes."

**Formule 3 — Feature cachée / info rare**
"La plupart des créateurs ne savent pas que Claude peut [action précise]."

**Formule 4 — Démo directe**
"Regarde ce que Claude vient de faire."
[montrer immédiatement le résultat — pas de blabla avant]

**Formule 5 — Problème nommé**
"Si tu galères encore avec [douleur précise de la cible], c'est parce que tu fais pas ça."

### Règles absolues pour tout hook
- ✅ Maximum 2 lignes à l'oral
- ✅ Minimum 1 chiffre précis si possible
- ✅ Se dit en < 3 secondes
- ✅ Pour la niche IA : toujours nommer l'outil (Claude, pas juste "l'IA")
- ❌ Jamais "Bonjour", "Salut", "Aujourd'hui on va voir"
- ❌ Jamais de question fermée (réponse oui/non)
- ❌ Jamais de hook vague ("l'IA c'est fou")

## 6. INSTAGRAM — ALGORITHME 2026

### Mécanisme de distribution
- Algo **user-first** (comme TikTok) : Instagram analyse les utilisateurs d'abord, matche le contenu
- **Groupe initial** : ~200 non-abonnés → données positives/neutres/négatives → push élargi si positif
- **Trust score** : chaque compte est noté (0-100). Actions négatives = distribution réduite
- ⚠️ Ne pas partager ses Reels à ses amis proches → biais négatif sur l'échantillon initial

### 3 métriques core Instagram
1. **Watch time** : temps total sur la vidéo
2. **Engagement rate** : ratio interactions / vues
3. **Watch time session** : est-ce que ta vidéo garde les gens sur Instagram plus longtemps ?

### Format optimal
- **Reels 60-90s** = format optimal → parfait pour cross-post depuis TikTok ✅
- Distribution Reels = 3 surfaces (Feed + Explore + Onglet Reels) vs 2 pour les posts

### Trial Reels (nouveau 2026)
- 5 Trial Reels/jour max
- 4 types batchables : self-intro / best recreation / low-hanging fruit / DM automation
- Ne jamais dupliquer le contenu à l'identique → recréer si tu reprends un sujet

### Upcycling
- Reposter le même contenu (refait) toutes les 90+ jours
- Minimum 3 tentatives par sujet avant de l'abandonner
