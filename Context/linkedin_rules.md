# RÈGLES LINKEDIN 2026 — Agent Marketing Kairos

## ALGORITHME — 3 PHASES DE DISTRIBUTION

1. **Phase test (60-120 min)** : montré à un petit échantillon — les commentaires dans cette fenêtre sont décisifs
2. **Reach élargie** : si engagement positif → distribution audience + hors-réseau
3. **Viralité** : reposts = carburant principal

→ Répondre à TOUS les commentaires dans la première heure = double le nombre de commentaires

## HIÉRARCHIE DES SIGNAUX

| Signal | Poids |
|---|---|
| Reposts | Le plus fort |
| Saves | Fort |
| Commentaires | = 10 likes |
| Likes | Faible |
| Lien externe dans le post | NÉGATIF — tue le reach |

## RÈGLE ABSOLUE : JAMAIS DE LIEN DANS LE POST
- Un lien externe = pénalité algo immédiate
- À la place : "Commente [MOT-CLÉ] et je t'envoie le lien en DM"
- Double bénéfice : signal commentaire fort + lead qualifié

## FORMAT DES POSTS

**Structure obligatoire :**
- Hook : 8-12 mots MAX. Pas d'emoji. Pas de hashtag. Phrase qui fait arrêter le scroll.
- Corps : ~150-200 mots. Retours à la ligne fréquents (lecture mobile).
- Pas de hashtags (devenus inutiles en 2026)
- Pas d'emojis dans le hook — possible dans le corps avec parcimonie

**Formats par performance :**
1. Carousel (fond sombre + dégradé) = format roi
2. Texte + photo de toi = humanise
3. Vidéo = bien
4. Texte seul = passable
5. Lien externe = à bannir

## TIMING DE PUBLICATION
- Matin : 7h-11h (idéal : 9h)
- Après-midi : 17h
- Meilleurs jours : Mardi > Lundi > Mercredi
- Éviter weekends

## 3 TYPES DE POSTS (rotation hebdomadaire)

**TYPE A — EXPERTISE IA (3-4 posts/semaine)**
Sujets : actus Claude/Anthropic, astuces techniques, démos d'agents, résultats chiffrés, comparaison avant/après, mini-tuto en steps, réfutation idée reçue sur l'IA.

**TYPE B — BUILDING IN PUBLIC / VIE (2 posts/semaine)**
Sujets : coulisses du business, leçon apprise cette semaine, erreur commise, "19 ans solo entrepreneur", process de construction, revenus/résultats (transparence partielle).

**TYPE C — LEAD MAGNET (1 post/semaine)**
Format : "Commente AGENT + enregistre ce post pour recevoir l'Agent TikTok IA gratuit en DM"
Objectif : commentaires massifs (signal fort) + leads qualifiés

## AUDIENCE CIBLE LINKEDIN
- Solopreneurs, indépendants, freelances francophones
- Créateurs de contenu TikTok/Instagram
- Petites agences
- Budget : 99-300€ justifiable
- Problème : perdent du temps sur les tâches répétitives

## RÈGLES DE CONTENU

**On parle de :**
- Agents natifs Claude (pas n8n/Make)
- Automatisation de la création de contenu
- Gain de temps précis et chiffré
- Building in public (19 ans, seul, France)
- Actus Claude/Anthropic

**On ne parle pas de :**
- Prix dans les posts (sauf post dédié)
- Concurrents spécifiques
- Sujets hors niche IA/automatisation

1. Démo agent en action (résultat visible)

3. Actualité Claude/Anthropic + mon avis
4. Erreur commune à éviter avec l'IA
5. Process building en public cette semaine

## SHADOW BAN — À ÉVITER

- Hashtags en masse

- Contenu hors niche
