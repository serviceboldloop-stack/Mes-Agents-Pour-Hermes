## TYPES DE CONTENU (à mixer chaque semaine)

### Type A — Clip démo depuis YouTube (prioritaire)
Le moment le plus impressionnant ou utile de la vidéo YouTube.
Règle : l'extrait doit avoir du sens seul, sans avoir vu la vidéo.
```
[Extrait du moment le plus fort de la démo]
→ CTA : "Tuto complet en lien dans ma bio."
```

### Type B — Actu IA (info de la semaine)
News Claude/IA + application directe pour la cible.
```
Hook : "Claude vient de sortir [feature]. Voilà ce que ça change pour [cible]."
Corps : 30-40s d'explication concrète
CTA : "Commente AGENT si tu veux [ressource liée]."
```

### Type C — Démo flash (le plus fort)
Montrer l'agent en action de A à Z en 45-60s.
```
Hook : résultat immédiat (l'output de l'agent) → puis expliquer comment
Corps : démo ultra compressée
CTA : 3 CTAs empilés
```

### Type D — Problème → Solution
Nommer le problème de la cible → montrer que l'agent le résout.
```
Hook : "Tu passes [X] heures à [tâche] chaque semaine ?"
Corps : "J'ai créé un agent Claude qui fait ça en [temps]."
CTA : "Commente AGENT."
```

### Type E — Avant/Après automatisation (gap concurrentiel — personne ne le fait)
Montrer la transformation chiffrée, pas les fonctionnalités.
```
[0-3s] HOOK : "Combien de temps tu passes sur [tâche connue] par semaine ?"
[3-10s] AVANT : "[X heures] sur [tâche] — chaque semaine — c'était mon cas."
[10-40s] AGENT : montrer l'agent Claude en action à l'écran
[40-55s] APRÈS : "[X minutes] maintenant. Temps récupéré : [Y heures/semaine]."
[55-70s] CTA : "Commente [MOT] si tu veux l'agent."
```
Note : c'est le format de transformation le plus puissant pour convaincre. Aucun concurrent ne l'utilise en France.

### Type F — ROI chiffré (gap concurrentiel)
Donner des métriques concrètes en € ou en heures. Pas de fonctionnalités, que des résultats.
```
Hook : "Ce que mon agent Claude a produit cette semaine : [chiffre surprenant]"
Corps : liste des métriques réelles (ex : "312 commentaires = 312 emails capturés")
CTA : "Commente [MOT] si tu veux le même système."
```

Si script YouTube disponible :
- Identifier les 3 moments les plus forts (démo, résultat, astuce)
- Pour chaque moment : est-ce qu'il tient seul en 30-60s ? Si oui → script TikTok clip

### Phase 2 — Écriture des scripts

**Pour chaque script, respecter la structure :**
```
[HOOK — 0-3s] : maximum 2 lignes, 1 chiffre si possible
[WTF — 3-8s] : révélation immédiate (résultat ou fait surprenant)
[DÉVELOPPEMENT — 8-45s] : valeur dense, phrases courtes, démo ou explication
[CTAs — 45-60s] :
  "Commente AGENT si tu veux [ressource]."
  "Enregistre ça — tu vas en avoir besoin."
  "Envoie ça à [profil cible qui a ce problème]."
```

**Règles strictes :**
- ✅ Hook en < 3 secondes à l'oral
- ✅ Nommer Claude (pas juste "l'IA") — différenciateur et SEO
- ✅ 3 CTAs empilés minimum
- ✅ "Commente [MOT-CLÉ]" = CTA N°1 absolu sur tous les scripts (le mot-clé varie selon la vidéo)
- ✅ Le mot-clé ManyChat doit être dit à l'ORAL ET dans la description (si seulement dans la description, l'automatisation ne se déclenche pas — erreur observée chez Ayoub dans ses anciens posts)
- ✅ Chaque vidéo = un mot-clé unique (ne jamais réutiliser le même mot-clé pour deux vidéos différentes)
- ✅ Phrases max 8-10 mots
- ✅ Durée cible : 55-75 secondes. Jamais dépasser 90 secondes sur IG/TikTok
- ✅ Musique de fond légère obligatoire (lo-fi, cinématique, suspense — jamais de silence)
- ✅ Sous-titres sur 100% des vidéos (70-80% des vues sans son)
- ✅ Signature verbale FIXE à la fin de chaque reel : **"C'était Lucas."** (règle STYLE-07 — jamais de variation)
- ✅ Montrer une interface à l'écran pendant la démo (jamais 100% face caméra sur du contenu tech)
- ❌ Jamais "Bonjour", "Salut", "Aujourd'hui on va voir"
- ❌ Jamais vente directe (prix, lien direct vers 99€) — d'abord valeur → ManyChat → email → séquence → vente
- ❌ Jamais demander les likes
- ❌ Jamais plusieurs CTAs dans une même vidéo (1 CTA = 1 action)

### Phase 2b — Carousel Instagram (si le sujet s'y prête)

Un carousel est pertinent si le sujet peut être découpé en liste (7-15 éléments), progression (étapes), ou comparatif. Priorité quand le script YT contient une liste ou un guide.

**Structure standard (12 slides) :**
```
Slide 1 — Cover
Fond uni couleur Kairos (ou fond blanc cassé)
[Chiffre en accent coloré] + [Titre en texte noir gras]
Exemple : "7 agents Claude qui travaillent pour toi 24h/24"
Sous-titre : "(à déployer immédiatement)"
Signature @handle en bas
Indicateur : "swipe →"

Slides 2 à 11 — Corps
Numéro en haut à gauche
Titre de l'élément en gras
2-3 lignes d'explication
Indicateur de progression en bas (points ou X/11)
Signature @handle en bas à droite

Slide 12 — CTA
Grand texte centré :
"Commente '[MOT-CLÉ]' pour recevoir [ressource] en PDF"
Décoration simple (étoile, ligne)
Signature @handle
```

**Règles carousel :**
- 12 slides = standard (jamais moins de 9)
- Musique de fond : ajouter au moment de la publication Instagram
- Republier en story dans les 2h après publication
- Technique du "cache" en story : couvrir la révélation → force le clic vers le carousel
- Commentaire épinglé sous le post : rappel du CTA + lien ressource

### Phase 4 — Messages DM CreatorFlow (OBLIGATOIRE)

Générer **4 variations** du message de réponse automatique au commentaire "AGENT".
Chaque variation doit :
- Être différente dans le ton (directe / chaleureuse / sobre / enthousiaste)
- Contenir le lien de la page de capture email liée à la ressource du post
- Faire 3-5 lignes max
- Être naturelle, pas robotique

Générer aussi **1 message ressource** pour l'envoi manuel en DM (plus long, avec prénom).

Sauvegarder dans `dm_messages.md` du dossier output courant.

Le fichier doit couvrir **4 plateformes** :
- Instagram → 4 variations DM (CreatorFlow)
- TikTok → réponse publique (TikTok Studio)
- YouTube Shorts → commentaire épinglé + description + réponse auto
- Message manuel (envoi DM à la main, toutes plateformes)

### Phase 5 — Ordre et timing de publication
Recommander l'ordre de publication sur la semaine :
- Jour 1 : clip le plus fort depuis YouTube (capitaliser sur la sortie YT)
- Jour 3 : actu IA ou démo flash
- Jour 5 : problème → solution ou deuxième clip YT
