## HOOKS YOUTUBE (titres + hooks d'intro)

### Formule 1 — Résultat + temps
```
Titre : "J'ai automatisé mes scripts TikTok avec Claude en 2h — voilà comment"
Intro : "Il y a 2 semaines, j'écrivais mes scripts TikTok à la main. Ça me prenait 1h par vidéo.
         Maintenant mon agent Claude le fait en 30 secondes. Je te montre exactement comment."
```

### Formule 2 — Problème résolu
```
Titre : "Fini les heures perdues à écrire du contenu — cet agent Claude fait tout"
Intro : "Si tu passes plus de 30 minutes à préparer chaque vidéo, c'est parce que tu fais pas ça."
```

### Formule 3 — Curiosité + secret
```
Titre : "La vraie raison pour laquelle les créateurs qui utilisent Claude publient 5x plus vite"
Intro : "Les créateurs qui publient le plus ne travaillent pas plus. Ils ont juste externalisé les tâches répétitives à Claude."
```

### Formule 4 — Tuto + chiffre
```
Titre : "Créer ton premier agent Claude en 15 minutes (même sans coder)"
Intro : "Dans 15 minutes, tu vas avoir ton premier agent Claude qui tourne. Même si t'as jamais touché à du code."
```

### Formule 5 — News + impact
```
Titre : "Cette nouvelle feature Claude change tout pour les créateurs de contenu"
Intro : "Anthropic vient de sortir [feature]. Voilà ce que ça change concrètement si tu crées du contenu."
```

## HOOKS À ÉVITER (dans cette niche)

```
❌ "L'IA c'est fou" → trop vague
❌ "Vous n'allez pas croire ce que j'ai découvert" → clickbait sans substance
❌ "Je vais vous montrer comment..." → trop formel, pas naturel
❌ "Aujourd'hui on va parler de Claude" → pas de curiosité
❌ "Bonjour, c'est Lucas" → personne ne connaît Lucas encore
❌ "Je suis solopreneur de 19 ans" → se présenter trop tôt = drop immédiat
```

## RÈGLES UNIVERSELLES DES HOOKS

1. **Jamais commencer par "Je"** — commence par le problème de l'audience ou le résultat
2. **1 chiffre précis si possible** — "30 secondes" > "très rapidement"
3. **Nommer Claude** — pas juste "l'IA" → différenciateur + SEO
4. **La tension doit se résoudre dans la vidéo** — pas de promesse non tenue
5. **Max 2 lignes à l'oral pour TikTok** — se dit en < 3 secondes
