# OUTPUTS — Dossiers de production

Chaque cycle de production crée un dossier :
```
[YYYY-MM-DD_Sujet-court]/
├── veille.md              ← sujets + actus IA identifiés
├── youtube_script.md      ← script complet + 3 options de titres
├── youtube_description.md ← description YouTube prête à copier
├── tiktok_scripts.md      ← 3-5 scripts TikTok/Instagram
├── newsletter.md          ← email hebdo + onboarding si nouveau lead
├── stories.md             ← séquence stories Instagram
├── analyse.md             ← analyse post-publication (J+7)
└── readme.md              ← récap + checklist publication
```

## Nommage
Format : `YYYY-MM-DD_Sujet-en-kebab-case`
Ex : `2026-05-19_Agent-TikTok-IA-Gratuit`

## Statuts
- ⬜ À produire
- 🎬 En production
- ✅ Publié
- 📊 Analysé
