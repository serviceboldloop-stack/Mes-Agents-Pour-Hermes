# SCRIPTS REELS — Semaine 1 (28 mai au 3 juin 2026)
## Format : face cam + démo screen | durée variable 25-60s selon démo | ManyChat activé

**Règle durée :** la vidéo dure exactement ce que la démo exige. Pas plus.
L'objectif n'est pas le maximum de vues. C'est le maximum de prospects (commentaires ManyChat → email).
Une vidéo de 30s vue à 90% de complétion bat une vidéo de 75s vue à 20%.

---

## RÈGLES DE TOURNAGE (rappel)

- Chaque reel = 1 seul mot-clé ManyChat (unique, jamais répété)
- Dire le mot-clé à VOIX HAUTE dans la vidéo + le mettre dans la description
- Structure adaptée à la durée : hook 2-3s / contexte 5-10s / démo 15-35s / CTA 8-10s
- Pas de signature verbale en fin de reel — le CTA suffit, couper après
- Musique de fond légère sur TOUTE la vidéo (lo-fi ou cinématique)
- Sous-titres sur 100% de la vidéo (brûlés au montage)
- Hashtags : 8-10 par post, toujours en fin de description, mix populaire + niche

## NOTES DE MONTAGE (valables pour tous les reels)

- Coupes rapides : max 3-4 secondes sans coupe
- Quand tu passes à la démo screen : transition propre (glissement ou cut)
- Fenêtre démo : prend les 3/4 de l'écran pendant la phase démonstration
- Son "pop" quand tu passes d'un résultat à l'autre
- Musique : lo-fi légère, pas plus de -20dB (voix doit dominer)
- Sous-titres : blancs, police bold, centrés, 2-3 mots max par ligne

## PATTERNS DE STYLE LUCAS (à maintenir dans tous les scripts futurs)

Appris des modifications manuelles de Lucas (2026-05-28) :
- Grammaire orale assumée : "je ne comprennait pas", "tu parle", "5 minute", "lui il créer" → ne pas corriger
- "Alors" comme transition naturelle
- CTAs courts et directs : "je te le donne" / "je te l'envoie" — pas d'explication
- Superlatifs modérés : "incroyable" plutôt que "indécente"
- Jamais de ChatGPT memory comme argument — techniquement inexact avec Codex/projets ChatGPT
- Sans agent dédié = le vrai problème (valable peu importe l'outil)
