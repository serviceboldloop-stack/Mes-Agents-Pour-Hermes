# PROMPT TEMPLATE — Générateur de Scripts TikTok/Instagram
## Ressource gratuite — à copier-coller dans Claude

---

## MODE D'EMPLOI

1. Copie tout ce prompt
2. Colle-le dans Claude (claude.ai ou Claude Desktop)
3. Remplis les 5 zones entre crochets `[...]`
4. Lance — tu reçois 3 scripts prêts à filmer

---

## LE PROMPT

```
Tu es un expert en scripts vidéo courts pour TikTok et Instagram Reels.

Génère 1 script de 55 à 75 secondes, prêt à filmer.

────────────────────────────────────
MES INFORMATIONS (à remplir)
────────────────────────────────────

Ma niche : [TA NICHE — ex : coaching business, cuisine healthy, e-commerce, fitness, photographie...]

Mon audience cible : [QUI TU CIBLES — ex : "entrepreneurs 25-35 ans qui veulent scaler leur business" / "mamans débordées qui cherchent des recettes rapides"]

Sujet de la vidéo : [LE SUJET EXACT — ex : "comment j'ai économisé 3h par semaine" / "ma routine matinale qui a tout changé" / "l'erreur que font 90% des débutants en X"]

Mon mot-clé ManyChat : [TON MOT-CLÉ EN MAJUSCULES — ex : SCRIPT / GUIDE / ASTUCE / RECETTE / ROUTINE]

Format choisi : [CHOISIS 1 SEUL — Démo live / Problème-Solution / ROI chiffré / Avant-Après / Révélation / Info/Actu]

────────────────────────────────────
RÈGLES DE STRUCTURE (ne pas modifier)
────────────────────────────────────

Chaque script suit cette structure exacte :

[HOOK — 0 à 3s]
→ 1 à 2 lignes maximum
→ Se dit en moins de 3 secondes
→ Commence par le résultat, le chiffre ou le problème
→ Jamais "Bonjour", "Salut", "Aujourd'hui on va voir", "Je m'appelle X"
→ 1 chiffre précis si possible (ex : "30 secondes", "3h par semaine", "7 scripts")

[RÉVÉLATION — 3 à 8s]
→ Le fait surprenant ou le résultat immédiat
→ 1 seule phrase courte
→ Doit donner envie de voir la suite

[DÉVELOPPEMENT — 8 à 45s]
→ Valeur concrète et actionnable
→ Phrases courtes : 8 mots maximum par phrase
→ Si démo à l'écran : décrire ce qui se passe visuellement
→ Pas de blabla, pas de rembourrage

[CTAs — 45 à 60s]
→ 3 CTAs empilés dans cet ordre exact :
   1. "Commente [MOT-CLÉ] et je te l'envoie."
   2. "Enregistre cette vidéo — tu vas en avoir besoin."
   3. "Envoie ça à [quelqu'un dans ton entourage qui a ce problème]."

────────────────────────────────────
RÈGLES DE TON (ne pas modifier)
────────────────────────────────────

- Phrases max 8-10 mots
- Grammaire orale acceptée — naturel > parfait
- Pas de superlatifs vides ("incroyable", "révolutionnaire", "game-changer")
- Direct et dense — zéro phrase de remplissage
- Parle à "tu", pas à "vous"

────────────────────────────────────
FORMAT DE LIVRAISON (ne pas modifier)
────────────────────────────────────

Pour chaque script, livrer :

SCRIPT — [FORMAT CHOISI]
Durée estimée : [X] secondes

[HOOK — 0-3s]
[texte]

[RÉVÉLATION — 3-8s]
[texte]

[DÉVELOPPEMENT — 8-45s]
[texte]

[CTAs — 45-60s]
[texte]

DESCRIPTION (2-3 lignes pour TikTok/Instagram) :
[texte]

HASHTAGS (8 à 10) :
[hashtags]

---

Génère le script maintenant.
```
